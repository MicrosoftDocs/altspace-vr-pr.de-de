# <a name="using-the-groups-feature-to-create-private-events"></a>Verwenden der Gruppenfunktion zum Erstellen privater Ereignisse

Unsere Gruppenfunktion bietet Hosts die Möglichkeit, einen VR-Bereich zu erstellen, der für eine genehmigte Liste von Gästen geöffnet ist. Hosts können alle altspacevr-Benutzernamen/-e-Mails, denen Sie Zugriff auf ihren Platz gewähren möchten, manuell hinzufügen. Im folgenden wird erläutert, wie Sie dieses Feature zum Hosten von Ereignissen, Besprechungen und Konferenzen verwenden können:

> [!NOTE]
> Bevor Sie auf "Gruppen" zugreifen können, müssen Sie sich für die frühzeitigen Zugriffs Features anmelden, indem Sie sich bei altvr.com anmelden, das Benutzerprofil auf der Website bearbeiten und den frühen Zugriff auswählen. Oder in der altspacevr-APP, Einstellungen, allgemein, am frühzeitigen Zugriffs Programm teilnehmen.

1. Navigieren Sie in Ihrem Browser zu,  [https://altvr.com](https://altvr.com)   und melden Sie sich bei Ihrem Konto an. 
2. **Weitere** Informationen finden Sie im Dropdown Menü.
3. **Gruppen** auswählen

![Altspacevr-Webansicht mit hervorgehobenem Pulldownmenü](images/group-features-img-01.png)

4. Auf der Seite Gruppen können Sie eine Liste der Gruppen sehen, zu denen Sie gehören, oder einen Administrator von. 

![Seite "altspacevr-webansichts Gruppen"](images/group-features-img-02.png)

5. Wählen Sie auf der Schaltfläche **Erstellen** aus, um eine neue Gruppe zu erstellen.
6. Füllen Sie die Felder aus, um die Gruppe zu erstellen.

![Seite "neue Gruppen Erstellung" mit Eingabefeldern](images/group-features-img-03.png)

> [!NOTE]
> Sie können mit dem Mauszeiger auf das grüne "?" , um zusätzliche Informationen zu den Feldern zu erhalten, in denen Sie enthalten sind.

7. Senden Sie eine e-Mail an alle Teilnehmer, und erstellen Sie mit der gleichen e-Mail-Adresse ein altspacevr-Konto. Sie können entweder ein altspacevr-Konto in-World oder auf unserer [Website](https://account.altvr.com/users/sign_up)erstellen.

## <a name="how-to-create-a-private-eventfrom-a-group"></a>Erstellen eines privaten Ereignisses aus einer Gruppe 

1. Wählen Sie auf der Seite Mitglied oder Administrator die Gruppe aus, auf die Sie zugreifen möchten. 
2. **Als Administrator** können Sie alle Optionen ändern, die Sie zuvor konfiguriert haben, um die Gruppe zu erstellen.Wechseln Sie zur Schaltfläche "Bearbeiten" auf der Seite der Gruppe, um darauf zuzugreifen.  
3. **Hinzufügen weiterer Administratoren für eine Gruppe:**   Beim bearbeiten können Sie weitere Benutzer als Administratoren hinzufügen, um die Verwaltung der Gruppe zu unterstützen.  
    * Dabei muss es sich um den altspacevr-Benutzernamen für den Benutzer und nicht um die Konto-e-Mail  
4. Wählen Sie **Ereignis erstellen** aus.  

![Erstelltes Ereignis](images/group-features-img-04.png)

## <a name="how-to-attend-a-group-event-as-an-invitee"></a>So nehmen Sie als einladende an einem Gruppen Ereignis Teil 

1. Öffnen Sie das Hauptmenü in der altspacevr-App:

2. Wählen Sie am unteren Rand des Menüs die Option **Ereignisse** aus.
3. Suchen Sie oben nach " **Meine Ereignisse** "
4. Suchen Sie das Ereignis anhand seines **namens** , der vom Ereignis Ersteller in der Hauptansicht mit den Kachel Bildern bereitgestellt wird.Wählen Sie diese Option aus, um das Ereignis einzugeben, wenn es aktiv ist.

![Altspacevr-App mit hervorgehobenem Ereignis Bereich](images/group-features-img-06.png)

## <a name="how-to-use-patreon-and-eventbrite-to-host-a-ticketed-vr-event"></a>Verwenden von patreon und eventbrite zum Hosten eines Ticketed VR-Ereignisses: 

Sie können patreon, eventbrite, verwenden, um getickerte Ereignisse in altspace zu erstellen.Sie müssen entweder eine Liste der e-Mail-Adressen Ihrer Benutzer erstellen oder diese Informationen von ihrer Zahlungsplattform exportieren.  

### <a name="creating-a-list"></a>Erstellen einer Liste

Verwenden Sie ein Tabellen Kalkulations Programm zum Erstellen einer einzelnen Spalte mit allen e-Mail-Adressen. Gehen Sie anschließend wie folgt vor:  

1. Erstellen oder bearbeiten Sie die Gruppe.  
2. Suchen Sie das Feld Member-e- **Mail** .  
3. Suchen Sie die Option **Datei auswählen** , um Ihre e-Mails aus einer CSV-Datei zu importieren.
4. Ihre Liste sollte mit einer e-Mail-Adresse pro Zeile importiert werden.

> [!NOTE]
> Wenn Sie auswählen, können Sie auch e-Mails kopieren und in das Feld einfügen, anstatt Sie aus einer CSV-Datei zu importieren.

### <a name="importing-a-list"></a>Importieren einer Liste

Plattformen wie patreon und eventbrite ermöglichen das Exportieren einer CSV-Datei (durch Trennzeichen getrennte Werte), die über eine dedizierte Spalte "e-Mail" verfügt.Sie können diese Spalte für die Verwendung auswählen.Nachdem Sie diese Informationen verfügbar gemacht haben, sind die Schritte größtenteils identisch, Sie sind:  

1. Erstellen oder bearbeiten Sie die Gruppe.  
2. Suchen Sie das Feld Member-e- **Mail** .  
3. Suchen Sie die Option **Datei auswählen** , um Ihre e-Mails aus einer CSV-Datei zu importieren.
4. Ihre Liste sollte mit einer e-Mail-Adresse pro Zeile importiert werden.

> [!NOTE]
> Wenn Sie auswählen, können Sie auch e-Mails kopieren und in das Feld einfügen, anstatt Sie aus einer CSV-Datei zu importieren.

### <a name="i-want-to-use-eventbrite-and-patreon-but-i-dont-see-where-to-get-the-email-addresses-for-my-groups"></a>Ich möchte eventbrite und patreon verwenden, aber ich sehe nicht, wo die e-Mail-Adressen für meine Gruppen zu finden sind.

**Patreon ** 

[Patreon](https://www.patreon.com/)   ist eine Mitgliedschafts Plattform, bei der die Erstellungs Funktion ein Abonnement Programm für Ihre Inhalte erstellen kann.Das patreon bietet mehrere Möglichkeiten zum Exportieren von Daten zu den Daten, die sich auf die CSV-Anleitungen konzentrieren. Sie finden dort die Hilfeseite für [diese Informationen,](https://www.patreon.com/portal/how-to/export-pledge-data)   oder Sie können die folgenden Anweisungen befolgen:  

1. Suchen Sie das Feld "Gönner" in der Seitenleiste des Entwicklers auf der linken Seite der patreon-Website.  
2. Erweitern Sie diese Option, um den Beziehungs-Manager zu suchen, und wählen Sie ihn aus.  
3. Am oberen Rand der Seite befindet sich eine Download Schaltfläche mit dem Namen CSV. Wählen Sie ihn aus.  
4. Daraufhin wird ein Windows-Dialogfeld angezeigt, in dem die Datei gespeichert wird.Speichern Sie es.  
5. Befolgen Sie die Anweisungen zum Importieren auf der Seite "altspacevr Group", um diese Datei zu verwenden.  

**Eventbrite ** 

[Eventbrite](https://www.eventbrite.com)   ist eine Ticket Website, mit der Benutzer Tickets für Ereignisse verwalten und erstellen können.Wie bei patreon gibt es viele Möglichkeiten, Teilnehmer-e-Mail-Listen zu generieren.Wir konzentrieren uns auf die CSV-Anleitungen. Sie finden dort die Hilfeseite für [diese Informationen](https://www.eventbrite.com/support/articles/en_US/How_To/how-to-edit-and-export-an-attendee-summary-report?lg=en_US)   , oder Sie können die nachfolgenden Anweisungen befolgen, um diese Daten zu extrahieren.  

1. Suchen Sie im Menü "Profil" das Menü "Ereignisse verwalten". 
2. Wählen Sie das Ereignis aus, in dem die e-Mail-Liste der Teilnehmer angezeigt werden soll 
3. Suchen Sie im Dashboard des Ereignisses die Kategorie mit dem Namen "allgemeine Berichte". 
4. Sie können auch "allgemeine Berichte" im Menü "linke Spalte" unter "Analyse-> Ereignisberichte" suchen.   
5. Wählen Sie "Teilnehmer Zusammenfassung" aus. 
6. Suchen Sie den Abschnitt "Berichts Ergebnisse". 
7. Wählen Sie die Schaltfläche "CSV" aus, um die Datei herunterzuladen. 

> [!IMPORTANT]
> Dies ist ein Beta Feature. Altspacevr ist eine kostenlose benutzerfreundliche Lösung und ist nicht verantwortlich für technische Probleme, die möglicherweise durch die Benutzeroberflächen auftreten, durch die Teilnehmer daran gehindert werden, auf getickte Ereignisse zuzugreifen.

# <a name="host-tools-overview"></a>Übersicht über Host-Tools

Wenn Sie ein Ereignis erstellen, sind Sie der automatische Host dieses Ereignisses. Die Rolle Host gewährt Ihnen Zugriff auf spezielle Features und Funktionen, die speziell erstellt wurden, um das Hosten eines virtuellen Live-Ereignisses einfach und einfach zu machen. Im Folgenden finden Sie eine ausführliche Übersicht über diese Features und Funktionen sowie weitere hilfreiche Hinweise zum Hosten von Ereignissen in AltspaceVR.

## <a name="understanding-host-tools"></a>Grundlegendes zu Hosttools

Die Hosttools werden unten rechts in Ihrem Sichtfeld angezeigt. Sie steht Ihnen sofort nach der Ereigniserstehung zur Verfügung, sodass Sie Ihr Ereignis jederzeit eingeben können (die Startzeit des Ereignisses gilt nicht für den Host) und sich vor showtime mit den Tools in Verbindung zu machen. Mit den Hosttools erhalten Sie Schnellklickzugriff auf Dinge wie das Stummschalten und Messaging Ihrer Zielgruppe, das Ein- und Ausschalten Ihrer Stageblocker, das Megaphoneing, sodass Ihre Gäste ihre Hände heben können, um Fragen zu stellen oder Feedback zu geben und vieles mehr.

![Erweitertes Menü "Hosttools" zum Anzeigen aller verfügbaren Aktionen](images/host-tools-img-01.png) 

### <a name="toggle-mute-all"></a>Alle umschalten

Indem Sie Ihre Teilnehmer stummschalten, können Sie ablenkende Töne einschränken und die Dinge in Ihrem Raum gut und still halten. Die Mute-Funktion wird so lange wie von einem Host erlaubt oder wenn der Host den Speicherplatz verlässt, dauern. Wenn ein Teilnehmer den Raum einlädt oder versucht, die Stummschaltung zu entmuten, erhalten die Teilnehmer eine Meldung auf dem Bildschirm, in der sie darüber erfahren, dass sie vom Host stumm geschaltet wurden. Wenn der Host dieMutung entlädt, erhalten die Teilnehmer eine Nachricht, die sie darüber ins Leben gibt, dass sie auch unveränderlich sind.

### <a name="message-all"></a>Message All

Hosts können eine Nachricht an alle Teilnehmer senden. Verwenden Sie eine unserer Kurznotizen, oder schreiben Sie eine originale. Nach dem Senden erhält jeder die Nachricht als Popupfenster.

### <a name="amplify-my-voice"></a>Amplify My Voice

Hosts können ihre Mikrofone verstärken, um sicherzustellen, dass jeder in Ihrem Ereignis das hören kann, was Sie sagen, unabhängig von der Entfernung.

### <a name="stage-blocking-onoff"></a>Phasenblockierung (Ein/Aus)

Entscheiden Sie, wann und wo Teilnehmer teilnehmen können. Entscheiden Sie sich dafür, den Blickpunkt zu befestigen, sich abwechselnd zu präsentieren oder alle Teilnehmer auf der Stage zu einer Gruppenparty einzulädnen.

> [!IMPORTANT]
> Nicht alle Welten enthalten einen Phasenblocker, also wählen Sie Ihre Ereignisvorlagen mit Bedacht aus. Machen Sie sich mit unseren Ereignisvorlagen für Testereignisse vertraut, bevor Sie Ihr offizielles Ereignis erstellen.

![Das Menü "Hosttools" wurde erweitert, um verfügbare Optionen mit hervorgehobener Blockblocker für die Umschaltphase anzeigen zu können.](images/host-tools-img-02.png)

## <a name="host-panel"></a>Hostbereich

Wenn Sie auf die Schaltfläche Hostbereich klicken, werden erweiterte Optionen für Ihre Ereignisse geöffnet.

![Geöffneter Hostbereich mit hervorgehobener Registerkarte "Gäste"](images/host-tools-img-03.png)

### <a name="guests-tab"></a>Registerkarte "Gäste"

Auf der Registerkarte Gäste können Sie die Gäste, Moderatoren und Hosts in Ihrem Raum nachverfolgen. Es handelt sich um eine interaktive Liste, mit der Sie eine mehrseitige Aufschlüsselung erhalten, wer in Ihrem Ereignis ist. Wenn Sie auf einen Gast klicken, erhalten Sie weitere Optionen (das Hamburger-Menü), mit dem Sie einzeln stummschalten und megaphonen können.

### <a name="moderation-tools"></a>Moderationstools

Als Ereignishost sind Sie für die Verwaltung der Benutzererfahrung anderer verantwortlich. Um dies effizient zu erreichen, benötigen Sie stabile Moderationstools. Als Host eines Ereignisses können Sie störende Personen stummschalten, Regel-Breakers aus Ihrem Ereignis (vorübergehend oder während Ihres Ereignisses) herausdraufen und anderen sogar die Moderator-Rolle zuweisen, um Hilfe bei der Verwaltung der Zielgruppe zu erhalten.

![Geöffneter Hostbereich mit hervorgehobener Registerkarte "Teilnahme"](images/host-tools-img-04.png)

## <a name="audience-participation-panel"></a>Teilnahmebereich der Zielgruppe

Der Teilnahmebereich soll Sie bei der Verwaltung der Interaktion und Teilnahme Ihrer Zielgruppe unterstützen. Hier können Sie die Option "Hand-Raises zulassen" aktivieren, um die Teilnahme der Zielgruppe zu ermöglichen. In FrontRow-Ereignissen müssen Sie on-air sein, um dies zu aktivieren. Andernfalls wird diese Schaltfläche als abgeblendet angezeigt. Wenn die Funktion "Hand-Hebung zulassen" einschaltet, wird im unteren rechten Bereich des Sichtfelds Ihrer Gäste die Schaltfläche "Raise Hand" (Hand erhöhen) angezeigt. Wenn ein Gast diese Schaltfläche auswählt, werden vorname und benutzername in Ihrem Teilnahmebereich angezeigt. 

Von hier aus können Sie sie aufrufen, megaphonen, stummschalten, veralten und – wenn es sich bei Ihrem Ereignis um ein Front Row-Ereignis handelt – sie "On Air" verwenden, um sie über mehrere Räume hinweg zu spiegeln. Damit Sie den Überblick behalten können, wird der Teilnahmebereich sogar rot angezeigt, wenn Sie Gäste haben, die darauf warten, dass sie aufgerufen werden, und Sie werden darüber informieren, wie lange jeder Ihrer Gäste gewartet hat.
 
![Geöffneter Teilnahmebereich der Zielgruppe mit hervorgehobener "Raise Hand"](images/host-tools-img-05.png)

# <a name="roles-in-events-worlds-groups-and-in-altspacevr"></a>Rollen in Ereignissen, Welten, Gruppen und in AltspaceVR

Rollen sind wie das Guthaben am Ende eines Films. Es gibt den ausführenden Producer, den Director, die Actors und sogar die Kontrolle über die Menge. Sie sind alle für etwas anderes verantwortlich, und alle haben eine wichtige Rolle zu spielen.

## <a name="roles-in-events"></a>Rollen in Ereignissen

Sie müssen zuerst Ihr Ereignis erstellen und anschließend die Rollen hinzufügen. Nachdem Sie Ihr Ereignis auf der Website erstellt haben, wählen Sie **BEARBEITEN** aus, um die Schaltfläche Kontextrollen **Benutzer hinzufügen zu** öffnen. Hier fügen Sie AltspaceVR-Benutzernamen hinzu, um Rollen für Ihr Ereignis zu geben.

### <a name="event-creator-roles"></a>Rollen des Ereigniserstellers

Wenn Sie ein Ereignis erstellen, erhalten Sie automatisch Zugriff auf die Administrator-, Host- und Moderatorrollen:

* **Administrator**
    - Alle Ereignisverwaltungstools über die Website. Sie können Bearbeiten, Endereignis, Auf Entwurf festlegen, Re-Import World, Duplikatereignis, Löschereignis und Als Hauptereignis hinzufügen (nur öffentliche Ereignisse).
    - Aktiviert den World Editor in Ihrem Ereignis: Umgebungsausschneiden, Flug umschalten, Umgebung umschalten, Bearbeitungsmodus und Editorbereich.

* **Host**
    - Aktiviert Hosttools in Ihrem Ereignis: Umschalten von "Mute-All", "Message All", "Amplify My Voice", "Go On-Air" (nur FrontRow-Ereignisse), "Toggle Stage Blocker" und "Host Panel".

* **Moderator** 
    - Ermöglicht es Ihnen, Benutzer aus Ihrem Ereignis zu stummschalten, zu senden oder zu starten, indem Sie entweder die Schaltflächen im Nametag dieses Avatars oder den Hosttools/Hostbereich/Gäste/Gast auswählen verwenden, um dies zu tun.

### <a name="contextual-roles"></a>Kontextbezogene Rollen

Verfügbar beim Bearbeiten des Ereignisses:

| Role | Berechtigungen |
|---|---|
| Host* | Alle oben genannten Möglichkeiten wie der Host, einschließlich Hosttools |
| Nur Megaphone | Gewährt diesem Benutzer Zugriff auf Hosttools, aber nur "Amplify My Voice" |
| Moderator ^ | Identisch mit der Moderatorrolle "Ereignisersteller" |
| Pilotphase | Fügt Flugfunktionen für diese Person hinzu. HINWEIS: Muss dies unter Einstellungen/Eingabe/Fly aktivieren. |
| Terraformer | Fügt der Benutzeroberfläche die Schaltfläche "Welt-Editor" hinzu. |

> [!NOTE]
> ^ = Jeder Benutzer mit diesen Rollen kann das Ereignis frühzeitig eingeben, indem er zu Events/My Events/YourEvent (Ereignisse/Meine Ereignisse/YourEvent) geht. Dies ist ideal für technische Überprüfungen und Prüfungen!

### <a name="mid-event-role-promotion"></a>Mid-Event Rollenaufstufung

Jeder Benutzer mit der Hostrolle kann Benutzern, die sich während des Ereignisses in Ihrem Ereignis befinden, Rollen zuweisen, indem er Hosttools/Hostbereich/Gäste/Gast auswählen verwendet:

* **Host machen:** Gibt diesem Benutzer die Hostrolle für den Rest des Ereignisses.
* **Make Moderator:** Gibt diesem Benutzer die Moderator-Rolle für den Rest des Ereignisses.

### <a name="adding-more-admins-to-your-event"></a>Hinzufügen von mehr Administratoren zu Ihrem Ereignis

Sie können andere Benutzer als Administratoren hinzufügen, um Ihr Ereignis zu verwalten. Möglicherweise sind Sie bereits damit beschäftigt, Ihr Ereignis zu hosten. Sie können dabei helfen, Rollen, Start-/Endzeiten oder Beschreibungen Ihres Ereignisses hinzuzufügen. Seien Sie vorsichtig, da sie alles auf Ihrer Ereigniswebseite bearbeiten können, einschließlich Ereignis löschen. Stellen Sie daher sicher, dass Sie allen Administratoren vertrauen, die Sie hinzufügen.

So fügen Sie einen Administrator hinzu:
1. Bearbeiten Ihres Ereignisses
2. Wählen Sie **Erweiterte Optionen aus.** Suchen Sie dann im Abschnitt Erweitert nach Administratoren.
3. Klicken Sie **auf Benutzer hinzufügen,** und fügen Sie ihren Benutzernamen hinzu. 

> [!NOTE] 
> Dadurch erhalten sie nur die Administratorrolle. Wenn Sie möchten, dass sie über andere Rollen (Host, Moderator) verfügen, müssen diese wie gewohnt unter Kontextrollen hinzugefügt werden.

### <a name="giving-everyone-in-your-event-a-role"></a>Allen In-Ihrem Ereignis eine Rolle geben

Einige Ereignisse profitieren, wenn alle Benutzer über Superleistung verfügen können! Wenn sich Ihr Ereignis beispielsweise in einem großen Bereich befindet, soll jeder die Möglichkeit haben, meine Stimme zu fliegt und zu verstärken, damit Sie sich alle von weit entfernt hören können. Um dies hinzuzufügen, bearbeiten Sie Ihr Ereignis, klicken Sie auf Erweiterte Optionen, und unter dem Abschnitt In VR ist standardbedingte Rollen. Rollieren Sie den grünen Kreis für alle Rollen, die Sie hinzufügen können. Für das obige Beispiel fügen Sie Folgendes hinzu: megaphone_only, Pilot.

> [!NOTE] 
> Jede Person muss dies unter Einstellungen/Eingabe/Fly aktivieren.

## <a name="roles-in-worlds"></a>Rollen in Welten

Sie müssen zuerst Ihre Welt erstellen und anschließend die Rollen hinzufügen. Nachdem Sie Ihre Welt auf der Website erstellt haben, klicken Sie auf die Schaltfläche BEARBEITEN, um die Schaltfläche Kontextrollen "Benutzer hinzufügen" zu öffnen. Hier fügen Sie AltspaceVR-Benutzernamen hinzu, um Rollen für Ihre Welt zu geben.

### <a name="world-creator-roles"></a>Rollen von Erstellern in der Welt

Wenn Sie eine Welt erstellen, erhalten Sie automatisch die folgenden Rollen:

* **Administrator** 
    - Alle Verwaltungstools der Welt über die Website. Sie können Bearbeiten, Anzeigen: Diagnose – Objekte – Vorlage – Sicherungen, Löschen und Feature My World verwenden.

* **Moderator** 
    - Ermöglicht es Ihnen, Benutzer aus Ihrer Welt zu stummschalten, zu senden oder zu starten, indem Sie entweder die Schaltflächen im Namenstag dieses Avatars oder den Hosttools/Hostbereich verwenden, um dies zu tun.

### <a name="contextual-roles"></a>Kontextbezogene Rollen

Verfügbar, wenn Sie Ihre Welt bearbeiten:

| Role | Berechtigungen |
|---|---|
| Nur Megaphone | Gewährt diesem Benutzer Zugriff auf Hosttools, aber nur "Amplify My Voice" |
| Moderator | Identisch mit der Moderator-Rolle des Erstellers der Welt |
| Musical Performer |Audio mit besserer Qualität für Dieb- oder Nachspielgeräte  |
| Pilotphase | Fügt Flugfunktionen für diese Person hinzu. HINWEIS: Muss dies unter Einstellungen/Eingabe/Fly aktivieren. |
| Host | Identisch mit der Hostrolle "Weltersteller" |
| Terraformer | Fügt der Benutzeroberfläche die Schaltfläche "Welt-Editor" hinzu. | 

### <a name="adding-more-admins-to-your-world"></a>Hinzufügen von mehr Administratoren zu Ihrer Welt

Bearbeiten Sie Ihre Welt, klicken Sie auf Erweiterte Optionen, suchen Sie im Abschnitt Erweitert für Administratoren nach, klicken Sie auf Benutzer hinzufügen, und fügen Sie ihren Benutzernamen hinzu. 

> [!NOTE] 
> Dadurch erhalten sie nur die Administratorrolle. Wenn Sie möchten, dass sie über andere Rollen (Host, Moderator) verfügen, müssen diese wie gewohnt unter Kontextrollen hinzugefügt werden.

## <a name="roles-in-groups"></a>Rollen in Gruppen

Sie fügen einer Gruppe keine Host- oder Moderator-Rollen hinzu, aber Sie können Administratoren hinzufügen, um Sie bei der Organisation Ihrer Gruppe zu unterstützen. Seien Sie vorsichtig, da sie Ihre Gruppe bearbeiten können, einschließlich Löschen. Stellen Sie daher sicher, dass Sie allen Administratoren vertrauen, die Sie hinzufügen.

### <a name="adding-more-admins-to-your-group"></a>Hinzufügen von mehr Administratoren zu Ihrer Gruppe

Sie müssen zuerst Ihre Gruppe erstellen und anschließend Administratoren hinzufügen. Nachdem Sie Ihre Gruppe auf der Website erstellt haben, klicken Sie auf die Schaltfläche BEARBEITEN, scrollen Sie nach unten, und Sie sehen die Schaltfläche Administratoren Benutzer hinzufügen, über die Sie Benutzernamen hinzufügen können.

## <a name="roles-in-altspacevr"></a>Rollen in AltspaceVR

* **Early Access Program**: Das Early Access Program bietet Ihnen Zugriff auf die neuesten Features, die sich noch in der Entwicklung befinden. Viele Features für World Building, Moveable Menus und Groups sind nur für Benutzer verfügbar, die sich in Early Access befinden. So können Sie sich entscheiden: Was ist das Early Access Program?

* **Worlds Beta:** Öffnen Sie in der AltspaceVR-App Menü/Einstellungen/Aktivieren der Aktivierung von Worlds Beta. Dadurch wird die Registerkarte WORLDS (BETA) im Hauptmenü aktiviert, damit Sie auf Ihre Welten zugreifen können.

* **Community Helper:** Vote-to-Suspend, Zugriff auf den Community Helpers-Kanal auf dem AltspaceVR-Diskothekenserver und Frühzeitiger Zugriff auf bestimmte Communityankündigungen. Weitere Informationen dazu, wie Sie ein Community Helper werden können: Community Helper Program

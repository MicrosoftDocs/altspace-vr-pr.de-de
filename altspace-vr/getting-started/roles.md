# <a name="roles-in-events-worlds-groups-and-in-altspacevr"></a>Rollen in Ereignissen, Welten, Gruppen und in altspacevr

Rollen sind wie das Guthaben am Ende eines Films. Es gibt den leitenden Producer, den Director, die Actors und sogar die Menschen Kontrolle. Sie sind für eine andere Aufgabe zuständig, die alle eine wichtige Rolle spielt.

## <a name="roles-in-events"></a>Rollen in Ereignissen

Sie müssen zuerst das Ereignis erstellen und dann die Rollen nach hinzufügen. Nachdem Sie das Ereignis auf der Website erstellt haben, wählen Sie **Bearbeiten** aus, um die Schaltfläche **Benutzer hinzufügen** von Kontext Rollen zu öffnen. Hier fügen Sie "altspacevr Benutzernamen" hinzu, um Rollen für das Ereignis anzugeben.

### <a name="event-creator-roles"></a>Rollen für Ereignis Ersteller

Wenn Sie ein Ereignis erstellen, erhalten Sie automatisch Zugriff auf die Rollen admin, Presenter und Moderator:

* **Administrator**
    - Alle Ereignis Verwaltungs Tools über die Website. Sie können das Ereignis bearbeiten, beenden, auf Entwurf festlegen, Re-Import Welt, duplizieren, Ereignis löschen und als Hauptereignis hinzufügen (nur öffentliche Ereignisse).
    - Aktiviert den World-Editor in Ihrem Ereignis: Umgebungs Clipping, umschalten, umschalten, Umgebung umschalten, Bearbeitungsmodus und Editor Panel.

* **Darsteller**
    - Aktiviert die Hosttools in Ihrem Ereignis: "stumm schalten-alle", "alle Nachrichten", "meine Stimme verstärken" (nur FrontRow-Ereignisse), "Phasen Blocker umschalten" und "Host Panel" umschalten.

* **Moderator** 
    - Ermöglicht es Ihnen, Benutzer aus dem Ereignis zu stumm schalten, zu richten oder zu starten, indem Sie entweder die Schaltflächen im nameTag dieses Avatars verwenden oder den Host Tools/Host Panel/Gäste/Gast auswählen, um das gleiche zu tun.

### <a name="contextual-roles"></a>Kontextabhängige Rollen

Verfügbar, wenn Sie das Ereignis bearbeiten:

| Role | Berechtigungen |
|---|---|
| Host ^ | Die gleichen Funktionen wie der obige Presenter, einschließlich der Host Tools |
| Nur megaphone | Ermöglicht dem Benutzer den Zugriff auf Host Tools, aber nur meine Stimme zu verstärken. |
| Moderator ^ | Identisch mit der Rolle "ereignisersteller-Moderator" |
| Pilotphase | Fügt Flug Funktionen für diese Person hinzu. Hinweis: diese Option muss in "Einstellungen/Eingabe/Betrieb" aktiviert werden. |
| Presenter ^ | Identisch mit der Rolle "ereignisersteller-Presenter" |
| Terraformer | Fügt die Schaltfläche "World Editor" ihrer Schnittstelle hinzu |

> [!NOTE]
> ^ = Jeder mit diesen Rollen kann das Ereignis frühzeitig eingeben, indem er zu "Events/My Events/YourEvent" geht. Dies eignet sich hervorragend für technische Prüfungen und Proben.

### <a name="mid-event-role-promotion"></a>Herauf Stufung der Mid-Event Rolle

Jeder Benutzer mit der Presenter-/Host-Rolle kann Benutzern, die sich während Ihres Ereignisses befinden, Rollen zuweisen, die Hosttools/Host Panel/Gäste/Gast auswählen:

* **Host erstellen**: gibt diesem Benutzer die Hostrolle für den Rest des Ereignisses.
* **Moderator erstellen**: gibt diesem Benutzer die Rolle "Moderator" für den Rest des Ereignisses.

### <a name="adding-more-admins-to-your-event"></a>Hinzufügen von weiteren Administratoren zu Ihrem Ereignis

Sie können weitere Benutzer als Administratoren hinzufügen, um Sie bei der Verwaltung Ihres Ereignisses zu unterstützen. Vielleicht sind Sie bereits mit dem Hosting Ihres Ereignisses beschäftigt und können dabei helfen, Rollen, Start-/Endzeiten oder Beschreibungen Ihres Ereignisses hinzuzufügen. Seien Sie vorsichtig, da Sie alles auf Ihrer Ereignis Webseite bearbeiten können, einschließlich des Lösch Ereignisses. Stellen Sie daher sicher, dass Sie alle Administratoren Vertrauen, die Sie hinzufügen.

So fügen Sie einen Administrator hinzu:
1. Ereignis bearbeiten
2. Wählen Sie **Erweiterte Optionen** aus. Suchen Sie anschließend im Abschnitt "Erweitert" nach Administratoren.
3. Klicken Sie auf **Benutzer hinzufügen** und fügen Sie seinen Benutzernamen 

> [!NOTE] 
> Dadurch erhalten Sie nur die Rolle "Administrator". Wenn Sie andere Rollen (Host, Presenter, Moderator) haben möchten, müssen diese unter Kontext Rollen wie "Normal" hinzugefügt werden.

### <a name="giving-everyone-in-your-event-a-role"></a>Allen Benutzern in Ihrem Ereignis eine Rolle geben

Einige Ereignisse sind von Vorteil, wenn alle Benutzer Superkräfte haben können! Wenn sich Ihr Ereignis z. b. in einem großen Bereich befindet, können Sie möchten, dass alle Benutzer in der Lage sind, ihre Stimme zu untersuchen und zu verstärken, damit alles von weitem Weg ist. Um dies hinzuzufügen, bearbeiten Sie das Ereignis, klicken Sie auf Erweiterte Optionen, und wählen Sie im Abschnitt in VR die standardmäßigen Kontext Rollen aus. Führen Sie einen Rollover des grünen Kreises für alle Rollen aus, die Sie hinzufügen können. Im obigen Beispiel würden Sie Folgendes hinzufügen: megaphone_only, Pilotprojekt.

> [!NOTE] 
> Jede Person muss dies in "Einstellungen/Eingabe/Betrieb" aktivieren.

## <a name="roles-in-worlds"></a>Rollen in Welten

Sie müssen zuerst Ihre Welt erstellen und dann die Rollen nach hinzufügen. Nachdem Sie Ihre Welt auf der Website erstellt haben, klicken Sie auf die Schaltfläche "Bearbeiten". Dadurch wird die Schaltfläche "Benutzer hinzufügen" der Kontext Rolle geöffnet. Hier fügen Sie altspacevr User Names hinzu, um Rollen für Ihre Welt zu geben.

### <a name="world-creator-roles"></a>World Creator-Rollen

Wenn Sie eine Welt erstellen, erhalten Sie automatisch die folgenden Rollen:

* **Administrator** 
    - Alle Welt Verwaltungs Tools über die Website. Sie können bearbeiten, anzeigen: Diagnostics – Objects – Template-Backups, DELETE und Feature My World.

* **Moderator** 
    - Ermöglicht es Ihnen, Benutzer aus ihrer Welt zu stumm schalten, zu richten oder zu starten, indem Sie entweder die Schaltflächen im nameTag dieses Avatars verwenden oder den Host Tools/-Host Panel verwenden, um das gleiche zu tun.

### <a name="contextual-roles"></a>Kontextabhängige Rollen

Verfügbar, wenn Sie Ihre Welt bearbeiten:

| Role | Berechtigungen |
|---|---|
| Nur megaphone | Ermöglicht dem Benutzer den Zugriff auf Host Tools, aber nur meine Stimme zu verstärken. |
| Moderator | Identisch mit der Rolle "World Creator Moderator" |
| Musikalischer Akteur |Besseres Audioqualität für Sänger oder Musiker, die Instrumente spielen  |
| Pilotphase | Fügt Flug Funktionen für diese Person hinzu. Hinweis: diese Option muss in "Einstellungen/Eingabe/Betrieb" aktiviert werden. |
| Darsteller | Identisch mit der Rolle "World Creator Presenter" |
| Terraformer | Fügt die Schaltfläche "World Editor" ihrer Schnittstelle hinzu | 

### <a name="adding-more-admins-to-your-world"></a>Hinzufügen von weiteren Admins zu ihrer Welt

Bearbeiten Sie Ihre Welt, klicken Sie auf Erweiterte Optionen, und klicken Sie dann im Abschnitt erweitert für Administratoren auf Benutzer hinzufügen, und fügen Sie Ihren Benutzernamen hinzu. 

> [!NOTE] 
> Dadurch erhalten Sie nur die Rolle "Administrator". Wenn Sie andere Rollen (Host, Presenter, Moderator) haben möchten, müssen diese unter Kontext Rollen wie "Normal" hinzugefügt werden.

## <a name="roles-in-groups"></a>Rollen in Gruppen

Sie fügen einer Gruppe keine Presenter-oder Moderator-Rollen hinzu, aber Sie können Administratoren hinzufügen, um Sie bei der Organisation Ihrer Gruppe zu unterstützen. Seien Sie vorsichtig, wenn Sie die Gruppe bearbeiten möchten, einschließlich DELETE. Stellen Sie also sicher, dass Sie alle Administratoren Vertrauen, die Sie hinzufügen.

### <a name="adding-more-admins-to-your-group"></a>Hinzufügen von weiteren Administratoren zu Ihrer Gruppe

Sie müssen zuerst die Gruppe erstellen und dann die Administratoren nach hinzufügen. Nachdem Sie die Gruppe auf der Website erstellt haben, klicken Sie auf die Schaltfläche Bearbeiten, Scrollen Sie nach unten, und Sie sehen die Schaltfläche Administratoren hinzufügen, wo Sie Benutzernamen hinzufügen können.

## <a name="roles-in-altspacevr"></a>Rollen in altspacevr

* **Frühes Zugriffs Programm**: mit dem frühen Zugriffs Programm erhalten Sie Zugriff auf die neuesten Features, die sich noch in der Entwicklung befinden. Viele Welt Erstellungs-, aktualisierbare Menüs und Gruppen Features stehen nur Benutzern zur Verfügung, die sich frühzeitig darauf zugreifen können. so können Sie sich entscheiden: Was ist das frühe Zugriffs Programm?

* **World Beta**: Öffnen Sie in der altspacevr-APP das Menü/die Einstellungen, und aktivieren Sie die Option zum Aktivieren der Beta Version. Dadurch wird die Registerkarte "Worlds (Beta)" in Ihrem Hauptmenü aktiviert, damit Sie auf ihre Welten zugreifen können.

* Communityhilfsprogramm **: Stimme** zum aussetzen, Zugriff auf den communityhilfsanchannel im altspacevr-Zwietracht-Server und frühe Zugriffe auf bestimmte communityankündigungen. Ausführlichere Informationen dazu, wie Sie zu einer communityhilfsanwendung werden können: Community Helper Program

# <a name="roles-in-events-worlds-groups-and-in-altspacevr"></a>Rollen in Ereignissen, Welten, Gruppen und in AltspaceVR

Rollen ähneln dem "Guthaben" am Ende eines Films. Es gibt den ausführenden Producer, den Director, die Actors und sogar die Kontrolle über die Menge. Sie alle sind für etwas anderes zuständig, und alle haben eine wichtige Rolle zu spielen.

## <a name="roles-in-events"></a>Rollen in Ereignissen

Sie müssen zuerst Ihr Ereignis erstellen und anschließend die Rollen hinzufügen. Nachdem Sie Ihr Ereignis auf der Website erstellt haben, wählen Sie **BEARBEITEN** aus, um die Schaltfläche Kontextbezogene Rollen **Benutzer hinzufügen** zu öffnen. Hier fügen Sie AltspaceVR-Benutzernamen hinzu, um Rollen für Ihr Ereignis zu vergeben.

### <a name="event-creator-roles"></a>Rollen des Ereigniserstellers

Wenn Sie ein Ereignis erstellen, erhalten Sie automatisch Zugriff auf die Administrator-, Host- und Moderatorrollen:

* **Administrator**
    - Alle Ereignisverwaltungstools über die Website. Sie können "Edit", "End Event", "Set to Draft", "Re-Import World", "Duplicate Event", "Delete Event" und "Add as Main Event" (Nur öffentliche Ereignisse) bearbeiten.
    - Aktiviert den World Editor in Ihrem Ereignis: Umgebungsclipping, Umschalten von Flight, Umschalten der Umgebung, Bearbeitungsmodus und Editor-Bereich.

* **Host**
    - Aktiviert Hosttools in Ihrem Ereignis: "Mute-All", "Message All", "Amplify My Voice", "Go On-Air" (nur FrontRow-Ereignisse), "Toggle Stage Blocker" und "Host panel".

* **Moderator** 
    - Ermöglicht ihnen das Stummschalten, Senden oder Starten von Benutzern aus Ihrem Ereignis, indem Sie entweder die Schaltflächen auf dem NameTag dieses Avatars verwenden oder den Hosttools/Hostbereich/Gäste/Gast auswählen verwenden, um dies zu tun.

### <a name="contextual-roles"></a>Kontextbezogene Rollen

Verfügbar, wenn Sie Ihr Ereignis bearbeiten:

| Role | Berechtigungen |
|---|---|
| Host* | Alle gleichen Funktionen wie host oben, einschließlich Hosttools |
| Nur Megaphone | Gewährt diesem Benutzer Zugriff auf Hosttools, aber nur "Amplify My Voice" |
| Moderator ^ | Identisch mit der Moderatorrolle des Ereigniserstellers |
| Pilotphase | Fügt Flugfunktionen für diese Person hinzu. HINWEIS: Muss in Einstellungen/Eingabe/Fly aktiviert werden |
| Terraformer | Fügt der Benutzeroberfläche die Schaltfläche "World Editor" hinzu. |

> [!NOTE]
> ^ = Jeder Benutzer mit diesen Rollen kann frühzeitig in das Ereignis eintreten, indem er zu Events/My Events/YourEvent (Ereignisse/Meine Ereignisse/Ihr Ereignis) geht. Dies eignet sich hervorragend für Technische Prüfungen und Übungen!

### <a name="mid-event-role-promotion"></a>Mid-Event Rollen heraufstufung

Jeder Benutzer mit der Hostrolle kann Benutzern, die sich in Ihrem Ereignis befinden, während Ihres Ereignisses Rollen zuweisen, indem er Hosttools/Hostbereich/Gäste/Gast auswählen verwendet:

* **Host erstellen:** Weist diesem Benutzer die Hostrolle für den Rest des Ereignisses zu.
* **Make Moderator:** Gibt diesem Benutzer die Moderator-Rolle für den Rest des Ereignisses.

### <a name="adding-more-admins-to-your-event"></a>Hinzufügen weiterer Administratoren zu Ihrem Ereignis

Sie können andere Benutzer als Administratoren hinzufügen, um Sie bei der Verwaltung Ihres Ereignisses zu unterstützen. Vielleicht sind Sie bereits damit beschäftigt, Ihr Ereignis zu hosten. Sie können Ihnen dabei helfen, Rollen, Start-/Endzeiten oder Beschreibungen Ihres Ereignisses hinzuzufügen. Seien Sie vorsichtig, da sie alles auf Ihrer Ereigniswebseite bearbeiten können, einschließlich Ereignis löschen. Stellen Sie daher sicher, dass Sie allen Administratoren vertrauen, die Sie hinzufügen.

So fügen Sie einen Administrator hinzu:
1. Bearbeiten Ihres Ereignisses
2. Wählen Sie **Erweiterte Optionen aus.** suchen Sie dann im Abschnitt Erweitert nach Administratoren.
3. Klicken Sie auf **Benutzer hinzufügen,** und fügen Sie dessen Benutzernamen hinzu. 

> [!NOTE] 
> Dadurch erhalten sie nur die Administratorrolle, wenn Sie möchten, dass sie über andere Rollen (Host, Moderator) verfügen, müssen diese wie gewohnt unter Kontextrollen hinzugefügt werden.

### <a name="giving-everyone-in-your-event-a-role"></a>Jedem in Ihrem Ereignis eine Rolle geben

Einige Ereignisse sind von Vorteil, wenn alle Benutzer über Superleistung verfügen können. Wenn sich Ihr Ereignis beispielsweise in einem großen Bereich befindet, möchten Sie möglicherweise, dass alle Benutzer die Möglichkeit haben, meine Stimme zu flyn und zu erweitern, damit Sie sich alle von weit her hören können. Um dies hinzuzufügen, bearbeiten Sie Ihr Ereignis, klicken Sie auf Erweiterte Optionen, und befindet sich im Abschnitt In VR unter Standardkontextrollen. Rollup des grünen Kreises für alle Rollen, die Sie hinzufügen können. Für das obige Beispiel fügen Sie Folgendes hinzu: megaphone_only Pilot.

> [!NOTE] 
> Jede Person muss dies in Einstellungen/Eingabe/Fly aktivieren.

## <a name="roles-in-worlds"></a>Rollen in der Welt

Sie müssen Zuerst Ihre Welt erstellen und anschließend die Rollen hinzufügen. Nachdem Sie Ihre Welt auf der Website erstellt haben, klicken Sie auf die Schaltfläche BEARBEITEN, um die Schaltfläche Kontextrollen "Benutzer hinzufügen" zu öffnen. Hier fügen Sie AltspaceVR-Benutzernamen hinzu, um Rollen für Ihre Welt zu vergeben.

### <a name="world-creator-roles"></a>World Creator-Rollen

Wenn Sie eine Welt erstellen, erhalten Sie automatisch die folgenden Rollen:

* **Administrator** 
    - All World-Verwaltungstools über die Website. Sie können Bearbeiten, Anzeigen: Diagnose – Objekte – Vorlage – Sicherungen, Löschen und Feature My World.

* **Moderator** 
    - Ermöglicht ihnen das Stummschalten, Senden oder Starten von Benutzern aus Ihrer Welt, indem Sie entweder die Schaltflächen auf dem Nametag dieses Avatars oder den Hosttools/Hostbereich verwenden, um dies zu tun.

### <a name="contextual-roles"></a>Kontextbezogene Rollen

Verfügbar, wenn Sie Ihre Welt bearbeiten:

| Role | Berechtigungen |
|---|---|
| Nur Megaphone | Gewährt diesem Benutzer Zugriff auf Hosttools, aber nur "Amplify My Voice" |
| Moderator | Entspricht der Moderatorrolle "World Creator" |
| Dies ist ein Performer |Bessere Audioqualität für Musik oder Wiedergabeinstrument  |
| Pilotphase | Fügt Flugfunktionen für diese Person hinzu. HINWEIS: Muss in Einstellungen/Eingabe/Fly aktiviert werden |
| Host | Entspricht der Hostrolle "World Creator" |
| Terraformer | Fügt der Benutzeroberfläche die Schaltfläche "World Editor" hinzu. | 

### <a name="adding-more-admins-to-your-world"></a>Hinzufügen weiterer Administratoren zu Ihrer Welt

Bearbeiten Sie Ihre Welt, klicken Sie auf Erweiterte Optionen, suchen Sie im Abschnitt Erweitert nach Administratoren, klicken Sie auf Benutzer hinzufügen, und fügen Sie ihren Benutzernamen hinzu. 

> [!NOTE] 
> Dadurch erhalten sie nur die Administratorrolle, wenn Sie möchten, dass sie über andere Rollen (Host, Moderator) verfügen, müssen diese wie gewohnt unter Kontextrollen hinzugefügt werden.

## <a name="roles-in-groups"></a>Rollen in Gruppen

Sie werden einer Gruppe keine Host- oder Moderator-Rollen hinzufügen, aber Sie können Administratoren hinzufügen, um Ihre Gruppe zu organisieren. Seien Sie vorsichtig, da sie Ihre Gruppe bearbeiten können, einschließlich Löschen. Stellen Sie daher sicher, dass Sie allen Administratoren vertrauen, die Sie hinzufügen.

### <a name="adding-more-admins-to-your-group"></a>Hinzufügen weiterer Administratoren zu Ihrer Gruppe

Sie müssen zuerst Ihre Gruppe erstellen und anschließend Administratoren hinzufügen. Nachdem Sie Ihre Gruppe auf der Website erstellt haben, klicken Sie auf die Schaltfläche BEARBEITEN, scrollen Sie nach unten, und sie sehen die Schaltfläche Admins Add User (Benutzer hinzufügen), auf der Sie Benutzernamen hinzufügen können.
